# Quality Checklist

Run through this checklist before returning CompanyFacts.

## Completeness
- [ ] company_name is the official name (not ticker symbol)
- [ ] sector is from the taxonomy (not free-form)
- [ ] recent_news has 3+ items
- [ ] financial_highlights has 2+ items with numbers
- [ ] key_events has 1+ items
- [ ] sources has 1+ valid URLs

## Accuracy
- [ ] Every news item has a date
- [ ] Financial numbers include units ($, %, B/M)
- [ ] YoY comparisons are directionally correct
- [ ] No claims without a corresponding source URL

## Freshness
- [ ] News items are from the past 30 days
- [ ] Financial data is from the most recent quarter
- [ ] No stale or outdated information mixed in
